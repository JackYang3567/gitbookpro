2、添加插件

在book.json中配置如下（这里用code插件作为例子）： 

{
    "plugins": [
        "code"
    ],
    "pluginsConfig": {
        "code": {
            "copyButtons": false
        }
    }
}

3、安装插件

可以选择下列三种方式中的一种进行安装

  • 1）、执行gitbook install

    这其实是安装book.json中的配置
     $ gitbook install ./

  • 2）、npm安装

    命令格式npm install gitbook-plugin-插件名字
    $ npm install gitbook-plugin-code

  • 3)、从GitHub下载源码，放到node_modules文件夹里
